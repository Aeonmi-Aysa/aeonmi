# Aeonmi Wiki

This wiki reflects the **current implementation reality** of Aeonmi.

## What Aeonmi is

Aeonmi is an AI-native language/runtime ecosystem implemented in Rust with:
- Native `.ai` execution (Lexer → Parser → AST → IR → VM)
- QUBE `.qube` circuit parsing/execution
- Mother AI embryo loop
- Glyph vault/identity system
- Mint metadata generation
- Shard self-hosting compiler workstream (in progress)
