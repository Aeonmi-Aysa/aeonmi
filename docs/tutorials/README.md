## Prerequisites

Before starting these tutorials, make sure you have:

1. **Aeonmi installed** - Follow the [Installation Guide](../README.md#installation)
2. **Basic Rust knowledge** - Variables, functions, and basic syntax
3. **Mathematical background** - Basic linear algebra helpful but not required

## Quick Start

If you're ready to jump in, start with the [Hello Quantum World](hello-world.md) tutorial:

```bash
# Create a new Aeonmi project
aeon new my-first-quantum-project
cd my-first-quantum-project

# Follow along with the tutorial
aeon run src/main.aeon
```

## Learning Tips

### 🎓 For Students
- Start with the basic tutorials and progress sequentially
- Experiment with the code examples
- Don't worry about understanding everything immediately
- Quantum mechanics is counterintuitive - this is normal!

### 👩‍💻 For Developers
- Focus on the programming patterns and API usage
- Try modifying the examples to see different behaviors
- Pay attention to error handling and best practices
- Consider quantum algorithms as new tools in your toolkit

### 🏫 For Educators
- Use the tutorials as structured lesson plans
- The "Experiments" sections work well as assignments
- Each tutorial builds on previous concepts progressively
- Code examples are designed to be easily modified for demonstrations
