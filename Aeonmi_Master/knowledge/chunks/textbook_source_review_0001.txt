PURPOSE: Frequency-tracking module. Implements an associative map over a flat
         array using the [key, count] * N layout (2 fields per record). Provides
         pat_add (insert/increment), pat_count (lookup), pat_top (argmax).
         No imports — fully self-contained.
SYNTAX DEMONSTRATED:
  - Flat-array record pattern: 2 fields per entry, index arithmetic
  - len(arr) / 2 to get record count
  - While loop with manual index:  while (i < n) { ... i = i + 1; }
  - Array index arithmetic:  let base = i * 2;
  - Conditional record update (found/not-found pattern)
  - Rebuilding array on update (immutable-style copy)
  - Empty array initializer:  return [];
  - Best-value tracking (argmax pattern)
KEY CODE EXAMPLES:
◯ pat_add⟨pat, key⟩ {

    let i = 0;
    let found = -1;
    while (i < n) {
        let base = i * 2;
        let k = pat.slice(base, base + 1).pop();
        if (k == key) { found = i; }
        i = i + 1;
    }
    if (found == -1) {
        pat.push(key);
        pat.push(1);
        return pat;
    }
    // rebuild with incremented count at found position

}
◯ pat_top⟨pat⟩ {
    let n = len(pat) / 2;
    if (n == 0) { return ""; }
    let best_key = "";
    let best_count = 0;
    while (i < n) {
        if (c > best_count) { best_key = k; best_count = c; }
        i = i + 1;
    }

}

=== FILE: Aeonmi_Master/aeonmi_ai/learn/reinforce.ai ===
PURPOSE: Strength-weighted reinforcement memory. Flat array layout:
         [key, val, strength] * N (3 fields per record). Implements
         reinf_touch (add/strengthen), reinf_decay (weaken all by rate),
         reinf_score (lookup strength). Memory that forgets over time.
SYNTAX DEMONSTRATED:
  - 3-field flat-array record layout
  - len(mem) / 3 record count
  - Strength arithmetic:  out.push(s * rate);
  - "touch" (seen again = strengthen) vs "decay" (time passes = weaken)
  - Upsert pattern: add if not found, strengthen if found
  - Initial strength of 1 on first observation
  - Rebuilding arrays immutably on update
KEY CODE EXAMPLES:
◯ reinf_decay⟨mem, rate⟩ {
    let n = len(mem) / 3;
    let i = 0;
    let out = [];
    while (i < n) {
        let base = i * 3;
        let k = mem.slice(base, base + 1).pop();
        let v = mem.slice(base + 1, base + 2).pop();
        let s = mem.slice(base + 2, base + 3).pop();
        out.push(k);
        out.push(v);
        out.push(s * rate);
        i = i + 1;
    }
    return out;
}
