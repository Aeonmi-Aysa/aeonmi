=== The Gate Descriptor Format ===

BODY: Every `quantum_run()` call takes a gate descriptor string as its argument.
Understanding this format is essential for writing quantum-enhanced agents.

The format is: `"n_q n_c shots n_ops  [op_type qubit1 qubit2] ..."`

n_q: number of qubits
n_c: number of classical bits
shots: measurement repetitions
n_ops: number of gate operations

Each gate operation is three numbers: op_type, qubit1, qubit2.
- op_type 0: Hadamard gate (qubit1 = target, qubit2 = -1)
- op_type 4: CNOT gate (qubit1 = control, qubit2 = target)
- op_type 7: Measure (qubit1 = qubit index, qubit2 = classical bit index)

CODE:
```
// Bell state (2-qubit): H(q0), CNOT(q0,q1), Measure(q0->c0), Measure(q1->c1)
let bell_desc = "2 2 1024 4  0 0 -1  4 1 0  7 0 0  7 1 1";

// GHZ state (3-qubit): H(q0), CNOT(q0,q1), CNOT(q0,q2), Measure all
let ghz_desc = "3 3 1024 6  0 0 -1  4 1 0  4 2 0  7 0 0  7 1 1  7 2 2";

// QRNG (single qubit): H(q0), Measure(q0->c0)
let qrng_desc = "1 1 100 2  0 0 -1  7 0 0";
```
