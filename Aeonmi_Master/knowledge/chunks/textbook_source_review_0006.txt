PURPOSE: Phase 3 Learning Loop. Mother accumulates signals, reinforces memory
         on correct predictions, decays stale patterns, and evolves its bias
         over multiple cycles. Demonstrates memory-with-strength as the core
         learning substrate.
SYNTAX DEMONSTRATED:
  - b_m_decay: multiply every strength field by rate (0.9 in main)
  - c_reinforce: PROCEED/ACCELERATE bump proceed_count + confidence_sum
  - c_bias: percent integer = proceed_count / total * 100
  - d_learn_cycle: one full cycle — decay → reinforce → store signal
  - Signal as integer verdict code
  - Integer percent bias (no floats needed)
  - Budget comment: ⍝ Standalone. 10 fns + main = 11 total (budget <= 13).

◯ b_m_decay⟨m, rate⟩ {

    }

◯ d_learn_cycle⟨m, verdict, conf⟩ {

=== FILE: aeonmi_ai/selfmodel/model.ai ===
PURPOSE: Phase 4 Self-Model. Mother tracks its own prediction accuracy,
         calibration (average confidence when correct), and drift (cumulative
         prediction error). Each cycle Mother predicts then observes, scoring
         itself. Pure introspection — the AI knowing how well it knows things.
SYNTAX DEMONSTRATED:
  - Self-model state via flat memory (same structure as external memory)
  - c_accuracy: correct * 100 / predictions, clamped
  - c_calibration: conf_sum / correct (avg confidence when right)
  - c_observe: records pred, outcome, conf; increments on hit
  - a_abs: absolute value helper for error calculation
  - d_self_report: comprehensive human-readable summary
  - String concatenation in print:  print("Accuracy     : " + acc + "%");
KEY CODE EXAMPLES:
◯ c_observe⟨m, pred, outcome, conf⟩ {
    ⍝ record one prediction/outcome pair
    let hit = 0; if (pred == outcome) { hit = 1; }
    if (hit == 1) {
        m = b_m_store(m, "correct",  corr + 1);
        m = b_m_store(m, "conf_sum", csum + conf);
    }
    let err = a_abs(pred - outcome);
    m = b_m_store(m, "drift", drift + err);
    return m;
}
◯ d_self_report⟨m⟩ {
    print("=== MOTHER SELF-MODEL ===");
    print("Accuracy     : " + acc + "%");
    print("Calibration  : " + cal + "  (avg conf when right)");
    print("Total drift  : " + drift + "  (prediction error sum)");
}
