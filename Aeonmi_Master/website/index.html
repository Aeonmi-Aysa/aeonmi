Your First Quantum Circuit

Create a Bell state (maximum quantum entanglement) in 15 lines 

⍝ Hello Quantum — Bell State Creation

function create_bell_state() {
    print("Creating Bell state: (|00⟩ + |11⟩)/√2");
    
    ⍝ Initialize quantum state
    let a00 = 1.0;
    let a11 = 0.0;
    
    ⍝ Apply Hadamard gate (superposition)
    a00 = 0.707;
    
    ⍝ Apply CNOT gate (entanglement)
    a11 = a00;
    
    print("✓ Qubits entangled!");
    return 0;
}

create_bell_state();

Result: Only |00⟩ and |11⟩ outcomes appear — proof of quantum entanglement
